REFLEKSI:
Tidak akan ada output di layar jika file models/kontak.py dijalankan secara langsung. Ini disebabkan fakta bahwa file tersebut hanya berisi definisi class Kontak, serta atribut dan metodenya, tanpa perintah yang dieksekusi secara langsung. Karena Python tidak memiliki perintah untuk memanggil fungsi atau membuat objek di luar class, ia hanya akan membaca dan memuat definisi class ke dalam memori. Dengan kata lain, file kontak.py berfungsi sebagai modul pendukung yang dapat diimpor dan digunakan oleh file utama, main.py. Di sana objek dibuat dan method tampilkan_info() digunakan untuk menampilkan hasilnya.
